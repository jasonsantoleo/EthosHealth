## How to Seed

Run the following command to seed these users into your database:

```bash
npm run seed:users
```

## Usage

1. Start your development server: `npm run dev`
2. Navigate to the home page
3. Enter any of the ABHA IDs above in the "Create HealthID" or "AI Recommendation" section
4. The system will recognize these pre-seeded users
